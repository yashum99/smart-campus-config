# Smart-Campus Config

This repository contains centralized configuration files for the Smart-Campus microservices project.

## 📁 Config Files
- user-service.properties
- event-service.properties
- notification-service.properties

## ⚙️ Usage
These configurations are consumed by the Spring Cloud Config Server to provide externalized configuration to all microservices.

Smart-Campus (Microservices Architecture)
